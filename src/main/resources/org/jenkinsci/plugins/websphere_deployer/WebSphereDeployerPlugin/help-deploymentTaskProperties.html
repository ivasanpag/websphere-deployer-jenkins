You can specify one or more deployment options using the following form. Eg: The task "MapSharedLibForMod" installs shared libraries in your application.
  • MapSharedLibForMod;sharedLibName;myModuleUri1=lib1+lib2;myModuleUri2=lib1
  • BindJndiForEJBMessageBinding;JNDI;EJB1Unique=jndiSpecification1


Each column to modify must be placed on a new line!
